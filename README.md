# NYC Soccer Matchday (MVP)

A lightweight MVP to help soccer fans in NYC find bars showing specific matches and see where the crowd is (“I’m going”).

## Current MVP
- Home → Results → Venue flow
- Venue pages at `/venue/[id]`
- Supabase connected (venues table seeded)

## Tech
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Supabase (Postgres)

## Run locally
```bash
npm install
npm run dev
