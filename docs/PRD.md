1. Problem

- Fans don’t know which NYC bars are showing a specific soccer match
- Hard to tell club-specific vs general sports bars
- Showing up to the wrong bar wastes time

2. Target users

- Soccer fans in NYC (primary)
- Visiting soccer fans in NYC
- Bar owners/managers (secondary)

3. Core user journeys

Fan

- Open website
- Select match
- View list of bars
- View bar details
- Click “I’m going”

Bar

- Log in
- Claim bar
- Post matchday update (showing/not showing)

4. MVP features (must-have)

- Web app (NYC only)
- Login (Google / magic link)
- Bar discovery by match
- Bar classification (general vs club-specific)
- “I’m going”
- Bar posting updates

5. Explicit non-goals

- No ratings or reviews
- No sound tracking
- No reservations or payments
- No mobile app

6. Success metrics

- Searches per day
- Venue page views
- “I’m going” clicks
- Bar updates posted
